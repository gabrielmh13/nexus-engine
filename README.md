# Nexus Engine

This is a study project for developing a game engine in C++.

## Project Structure

- **engine/**: The core engine module, compiled as a static library.
- **game/**: The game executable that utilizes the engine library.

## Requirements

- CMake 3.20+
- C++ compiler with C++23 support (e.g., GCC 12+, Clang 15+, MSVC 19.34+)

## How to Build

```bash
cmake -B build
cmake --build build
```
